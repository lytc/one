before(String|Array|Node|NodeList|HtmlCollection el) => self

Using the following HTML:

1<div>
2    <div class="foo">item</div>
3    <div class="foo">item</div>
4    <div class="foo">item</div>
5</div>

Insert before each div.foo:

1$('.foo').after('<p>test</p>')

Each div.foo element gets the new content:

1<div>
2    <p>test</p>
3    <div class="foo">item</div>
4    <p>test</p>
5    <div class="foo">item</div>
6    <p>test</p>
7    <div class="foo">item</div>
8</div>
